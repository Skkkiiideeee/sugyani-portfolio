Automated fresh or rotten fruit detection system

A deep learning-based image classification model to detect fresh vs rotten fruits using ResNet50. The model can classify fruit images into fresh or rotten categories, providing real-time classification when integrated into an Android application.

Tech Stack

Python, OpenCV (Haar Cascade), Keras, Pandas, NumPy, Scikit-learn, Seaborn

Tools

Google Colab, Jupyter Notebook

Summary

This project presents an advanced computer vision solution for automated fruit quality assessment. Leveraging a ResNet50-based Convolutional Neural Network, the system classifies fruits as "rotten" or "fresh" to support robotic sorting in industrial environments. The model was trained on two datasets (3.2k and 10.5k images) and achieved up to 92.3% accuracy through rigorous data augmentation and class balancing. Optimizations reduced inference time by 25%, enabling efficient, real-time operation on conveyor-based robotic platforms. The modular design allows for easy adaptation to new fruit types and integration into existing agricultural automation systems.

  • Trained on large, diverse datasets with advanced augmentation for improved generalization.
  • Achieved high accuracy in distinguishing between fresh and rotten fruits.
  • Reduced model inference time by 25% for real-time deployment.
  • Designed for seamless integration with robotic and conveyor systems.
  • Codebase is adaptable for other classification tasks in agriculture and food processing.

Future Prospects & Enhancements

The fruit detection system is built for scalability and continuous improvement. Planned and potential enhancements include:

  • Multi-Class Classification: Extend detection to multiple fruit types and additional quality grades (e.g., overripe, bruised).
  • Edge Deployment: Optimize models for deployment on embedded systems (Raspberry Pi, NVIDIA Jetson) for on-site processing.
  • Real-Time Conveyor Integration: Integrate with PLCs and industrial robots for fully automated sorting lines.
  • Defect Localization: Implement bounding box regression to localize and highlight rotten areas on fruits.
  • Cloud-Based Analytics: Aggregate sorting data for trend analysis and supply chain optimization.
  • Mobile App Interface: Develop a companion app for remote monitoring and manual override.
  • Self-Learning System: Allow the model to improve with continuous data collection from production lines.
