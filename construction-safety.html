Construction Site Safety Detection System

A AIML project built on YOLOv8

Tech Stack

Python, OpenCV, TensorFlow, Keras, YOLOv5

Tools

Google Colab, Jupyter Notebook

Summary

This project focuses on enhancing safety at construction sites by leveraging computer vision and deep learning techniques. Using YOLOv5, the system is trained to accurately detect the presence of essential safety gear such as helmets and vests, as well as identify unauthorized personnel in restricted areas. The dataset was carefully curated and augmented to ensure robustness across diverse site conditions, resulting in a model that achieves over 91% accuracy in controlled tests. While real-time detection via live camera feeds is a planned enhancement, the current implementation processes images and pre-recorded videos, laying a strong foundation for scalable, automated safety monitoring. The solution is designed to be modular and extensible, making it suitable for integration with IoT devices and web-based dashboards in future iterations.

  • Trained on a comprehensive safety gear dataset with advanced augmentation for real-world variability.
  • Achieved high detection accuracy for helmets, vests, and unauthorized access in test environments.
  • Implemented with YOLOv5 for fast and efficient object detection.
  • Current system supports batch processing of images and videos; real-time detection is planned for upcoming releases.
  • Code and models are portable for deployment on cloud or edge devices.

Future Prospects & Enhancements

The Construction Site Safety Detection system is architected for future scalability and feature expansion. Planned enhancements include:

  • Real-Time Detection: Integrate live camera feed processing for instant safety compliance monitoring on active sites.
  • Automated Alert System: Trigger immediate notifications (SMS, email, or app alerts) to supervisors when safety violations or unauthorized access are detected.
  • Hard Hat and Vest Color Classification: Distinguish between different worker roles based on gear color for advanced analytics and compliance tracking.
  • Facial Recognition Integration: Identify and log authorized personnel, further reducing the risk of unauthorized access.
  • Web Dashboard & Analytics: Visualize safety incidents, compliance rates, and trends with interactive charts and reports.
  • Edge Deployment: Optimize models for deployment on low-power edge devices such as Raspberry Pi or NVIDIA Jetson for on-site processing.
  • Multi-Camera Support: Monitor large construction sites by aggregating feeds from multiple cameras.
  • Integration with Site Access Control: Automate gates or turnstiles based on detection results for seamless site management.

Potential Use Cases

  • Construction Companies: Automate safety compliance checks and reduce manual supervision costs.
  • Industrial Plants: Monitor PPE (Personal Protective Equipment) adherence in hazardous environments.
  • Event Venues or Restricted Zones: Prevent unauthorized access and ensure safety regulations are followed.
  • Government Safety Audits: Provide verifiable data for regulatory compliance and reporting.

Broader Impact

  • Reduce Workplace Accidents: Proactively identify safety violations to minimize injuries and fatalities.
  • Promote AI Adoption in Construction: Demonstrate practical benefits of AI in traditional industries.
  • Empower Safety Officers: Free up human resources for higher-level safety planning and response.
  • Open Source Collaboration: Encourage contributions from the global developer and safety community.

Deployment & DevOps Readiness

  • Containerization (Planned): Package the detection system in Docker containers for easy deployment across environments.
  • CI/CD Pipeline (Planned): Automate testing and deployment workflows using GitHub Actions or Jenkins.
  • Cloud & Edge Compatibility: Design supports both centralized cloud processing and local edge deployments.
